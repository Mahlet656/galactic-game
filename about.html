PROJECT OVERVIEW

Nebula Blade is a futuristic interactive web experience built using modern frontend technologies. The project combines immersive UI design, layered animations, DOM-based interaction systems, and responsive gameplay-inspired architecture.

Designed with a cinematic sci-fi aesthetic, the application focuses on atmosphere, user immersion, and interactive storytelling through motion and visual depth.
